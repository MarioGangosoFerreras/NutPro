# 🥗 NutPro

¡Bienvenido a **NutPro**! Una aplicación moderna y eficiente desarrollada con **Angular** para la gestión nutricional.

---

## 🛠️ Requisitos Previos

Antes de empezar, asegúrate de tener instalado lo siguiente:

*   **Node.js**: [Descargar aquí](https://nodejs.org/es/download) (Se recomienda la versión LTS).
*   **Angular CLI**: Instálalo globalmente ejecutando:
    ```bash
    npm install -g @angular/cli
    ```

---

## 🚀 Instalación y Configuración

Sigue estos pasos para poner en marcha el proyecto en tu máquina local:

1.  **Clona el repositorio** (o descarga el código).
2.  **Instala las dependencias** del proyecto:
    ```bash
    npm install
    ```
3.  **Inicia el servidor de desarrollo**:
    ```bash
    ng serve
    ```
4.  **Accede a la app**: Abre tu navegador en `http://localhost:4200/`. La aplicación se recargará automáticamente si realizas cambios en el código.
